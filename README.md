# Blogging

Small collection if simple tools I use to help when writing [my
blog](https://github.com/davep/davep.github.com).

Note that I always write my posts on macOS so `pbcopy` is assumed to be in
the environment. Note that the tools also assume that my blog sources live
in `~/write/davep.github.com/`.

My blog is generated using [BlogMore](https://blogmore.davep.dev/).

## blogattach

When run it creates the attachment directory for today. It also places the
path to the directory in the clipboard.

## bloginc

Running `bloginc {file}` will place the Markdown for including the given
file, as an image, into a Markdown document, into the clipboard.

## blogserve

When run this starts up a locally-served copy of my blog for
testing/reading.
